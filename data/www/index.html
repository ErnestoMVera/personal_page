Projects

Fast Fourier Transform (FFT) signal analyzer and image compression tool
My own implementation of the Fast Fourier Transform ready to be applied on signals for further analysis or as an image compression tool which works based on the concept of energy conservation on signals stated in the Parseval's Theorem. This principle serves as the basis of one of the key components of the JPEG compression format which is the most common compression format for photos.

This project is written in C++ using only the standard library and a make build system, as well as several tests scripts. The code can be found in the github repo and you can see the technical details and results of the project here.
On-road video classifier
A tool for a human to classify visual attention points (such as looking to the road, right, left or lap) manually by looking at a video footage of an on-road driving test conducted for my master's thesis.

The project is written in C++ and depends on the OpenCV and Tesseract OCR libraries. The project is build using a CMake build system and currently only tested for linux x86. The code is available in the github repo. Learn more here.
